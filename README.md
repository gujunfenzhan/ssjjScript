# 生死狙击挂机脚本

这是一个生死狙击的挂机脚本,需要你在script文件夹下编写你的动作

* 目前脚本可支持操作

```
Sleep [暂停秒数]
KeyDown [按什么键] 
KeyUp [松开什么键]
MouseDown [按下鼠标LEFT或RIGHT]
MouseUp [松开鼠标LEFT或RIGHT]
MouseMove [相对移动X坐标] [相对移动Y坐标]
MouseAbsoluteMove [绝对移动X坐标] [绝对移动Y坐标]
WhenSimilarity [相似度判断条件] [匹配图片位置] [执行的脚本文件]
```

* 注:脚本里已经有工业7的挂机脚本和工业区急速脚本
